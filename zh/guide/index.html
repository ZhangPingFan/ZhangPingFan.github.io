# 为什么写博客？

我从小就是个喜欢捣鼓的人，所以面对新鲜事物，总有一颗探索的心。

对于技术也不例外，常常会为了解决各种小问题而开发一套程序或工具，坚信科技改变世界。

身处于信息时代的浪潮之中，我们是幸运的，但也因此需要持续摸索，才能跟上脚步走在前沿。

当然，也要对成熟的框架思想进行融会贯通，取其精华，化为己有。

所以决定开启我的博客之旅，温故且知新，分享自己的学习收获，也借此记录自己的成长。
